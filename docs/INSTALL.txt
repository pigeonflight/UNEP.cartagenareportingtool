UNEP.cartagenareportingtool Installation
----------------------------------------

To install UNEP.cartagenareportingtool using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``UNEP.cartagenareportingtool`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        UNEP.cartagenareportingtool
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
